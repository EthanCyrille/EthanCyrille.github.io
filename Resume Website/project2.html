Celestica – Starlight Sanctum

Resume Project 1 Project 2

A cosmic jewelry immersive design experience

Description:

Celestica's Starlight Sanctum is a visual and interactive representation of the Cosmic Jewelry Collection. The project was designed to immerse viewers in the thematic ideals of the collection, highlighting the unique textures, forms, and ethereal aesthetics of the jewelry pieces.

Objective:

The objective was to create an immersive digital experience using Blender and web-based interactive elements. This involved modeling jewelry pieces, animating them, and integrating interactive elements to simulate real-world viewing experiences. The project also explored combining animation with basic coding for interactive effects.

Skills Learned:

  • 3D modeling and high-fidelity rendering in Blender
  • Rigging and animating digital objects for realistic motion
  • Integrating animations with basic coding for web presentation
  • UI/UX layout and visual storytelling for web
  • Designing immersive thematic experiences
  • Attention to lighting, texture, and color for aesthetic impact
  • Workflow optimization for digital assets
  • Collaborative project planning and iterative design improvement
  • File management and preparation for web deployment
  • Combining art direction with technical animation techniques
